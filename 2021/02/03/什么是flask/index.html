什么是flask

发表于 2021-02-03 更新于 2021-02-04 分类于 Flask框架 阅读次数： 阅读次数： Valine：
本文字数： 879 阅读时长 ≈ 1 分钟

flask在架设网站的结构中，属于一个代码框架，用于处理一系列的逻辑事项，完成路由分发(及根据前端请求，找到对应的视图函数)，其他的工作如报文打包回传等，都有服务器进行完成。

代码详解

最简单的flask代码实现
‘’’python

from flask import Flask

app = Flask(__name__,
            static_url_path="/python",
            static_folder="static",
            template_folder="templates"
            )

@app.route("/")
def index():
    """定义的试图函数"""
    return "hello"

if __name__ == '__main__':
    app.run()

'''

1.首先关于导入模块

最好是导入具体的包，如：
from flask import Flask
而不是所有的一次性导入，如：
from flask import *

2.关于name
这里的name表示的是整个模块的位置。

3.关于静态文件
默认情况下，项目的静态文件会放到默认的这个模块的下是static文件下，但是你也可以改变，可以通过static_folder参数来进行指定。

4.关于模板文件

默认情况下所有的模板文件会放到templates文件夹下，如果要改变，可以通过template_folder进行指定。

5.关于url访问路径
默认情况下我们在前端会通过如http://127.0.0.1:5000/static/index.html这样的方式访问后端，这里的static就是默认的url路径，如果要改变成http://127.0.0.1:5000/python/index.html，可以通过改变参数static_url_path=”/python”，来指定，虽然改变了前端访问的路径，但是此时的静态文件，还是存在原来的static文件夹下。
